# Code of Conduct

The `staticweaver` project adopts the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

The full text is available at the link above. A short summary:

- Be respectful. Disagreements happen — focus on the work, not the person.
- Assume good intent. Ask before escalating.
- No harassment, personal attacks, or discriminatory language.
- Keep project spaces (issues, PRs, discussions, commit messages) on-topic
  and constructive.

## Scope

This code applies to all project spaces — the GitHub repository, issues,
pull requests, discussions, release notes, and any other communication
channel tied to `staticweaver`. It also applies when an individual is
officially representing the project in public spaces.

## Reporting

Report violations to the maintainer via:

- A private GitHub Security Advisory for anything sensitive:
  <https://github.com/sebastienrousseau/staticweaver/security/advisories/new>
- Email: <sebastian.rousseau@gmail.com>

Reports are handled confidentially. Expect an acknowledgement within
72 hours and a resolution within 7 days for straightforward cases.

## Enforcement

The maintainer may remove, edit, or reject contributions, issues, and
comments that do not align with this code, and may temporarily or
permanently ban any contributor whose behaviour is deemed inappropriate,
threatening, or harmful.

Enforcement decisions follow the Contributor Covenant
[community impact guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines).
